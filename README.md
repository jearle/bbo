# cd-product-api

## Quick Start
```sh
git clone git@github.com:rcanalytics/cd-product-api.git
cd cd-product-api
git checkout develop
nvm use
yarn install
cd packages/server
yarn start
```

## Build Production
```sh
nvm use
yarn install
cd packages/server
yarn build:production
```
